Ejercicios de Css

Inicio Ejercicios de Css

Práctica de Css con los ejercicios que se encuentran en W3School

Tema: Css Syntax

1. Change the color of all p elements to "red".


                <head>
                  <style>
                      p{ color:red; }
                  </style>
                </head>

2. Change the color of the element with id="para1", to "red".


                <head>
                  <style>
                      #para1{ color:red; }
                  </style>
                </head>

Tema: Css How to

3. Add an external style sheet with the URL: "mystyle.css" .


                <head>
                  <style>
                    <link rel="stylesheet" href="mystyle.css" type="text/css">
                  </style>
                </head>

4. Set "background-color: linen" for the page, using an internal style sheet.


                <head>
                  <style>
                    body{ background-color: linen; }
                  </style>
                </head>

Tema: Css Background

5. Set "paper.gif" as the background image of the page.


                <head>
                  <style>
                    body { background-image: url("paper.gif"); }
                  </style>
                </head>

6. Specify that the background image should be shown once, in the top right corner.


                <head>
                  <style>
                    body{ 
                      background-image: url("img_tree.png");
                      background-repeat: no-repeat;
                      background-position: right top;
                    }
                  </style>
                </head>

Tema: Css Border

7. Set a "4px", "dotted" border for p


                <head>
                  <style>
                    p{
                      border-style: dotted;
                      border-width: 4px;
                    }
                  </style>
                </head>

8. Change the 3 border properties, so that they only show the border on the top side.


                <head>
                  <style>
                      p {
                        border-top-style: dotted;
                        border-top-width: 4px;
                        border-top-color: red;
                      }
                  </style>
                </head>

Tema: Css Margin

9. Set all margins for h1 to "25px".


                <head>
                  <style>
                    h1 {
                      background-color: lightblue;
                      margin: 25px;
                    }
                  </style>
                </head>

10. Use the margin property to center align the h1 element.


                <head>
                  <style>
                    h1 {
                      background-color: lightblue;
                      width: 300px;
                      margin: auto;
                    }
                  </style>
                </head>

Tema: Css Padding

11. Set the top padding of p to "30px".


                <head>
                  <style>
                    p {
                      background-color: lightblue;
                      padding: 30px
                    }
                  </style>
                </head>

12. Use the padding property to set the top and bottom paddings for p to "25px", and left and right paddings to "50px".


                <head>
                  <style>
                    p {
                      background-color: lightblue;
                      padding: 25px 50px;          
                    }
                  </style>
                </head>

Tema: Css height/Width

13. Set the height of h1 to "100px".


                <head>
                  <style>
                    h1 {
                      background-color: lightblue;
                      height: 100px;
                    }
                  </style>
                </head>

14. Set the width of h1 to "50%".


                <head>
                  <style>
                    h1 {
                      background-color: lightblue;
                      width: 50%;
                    }
                  </style>
                </head>

Tema: Css Box Model

15. Set the width of the div to "200px".


                <head>
                  <style>
                    div {
                      background-color: lightblue;
                      width: 200px;
                    }
                  </style>
                </head>

16. Set the border of the div to "25px solid navy".


                <head>
                  <style>
                    div {
                      background-color: lightblue;
                      width: 200px;
                      padding: 25px;
                      border: 25px solid navy;
                    }
                  </style>
                </head>

Tema: Css Outline

17. Set a "solid", "5px" outline for p


                <head>
                  <style>
                    p {
                      outline-style: solid;
                      outline-width: 5px;
                    }
                  </style>
                </head>

18. Set the outline color for p to "green".


                <head>
                  <style>
                    p {
                      outline-style: solid;
                      outline-width: 4px;
                      outline-color: green;
                    }
                  </style>
                </head>

Tema: Css Text

19. Set the text color for the page to "red", and the text color for h1 to "blue".


                <head>
                  <style>
                    body {
                      color: red;
                    }
                    
                    h1 {
                      color: blue;
                    }
                  </style>
                </head>

20. Style text in h1 to uppercase letters, and text in p to capitalized letters.


                <head>
                  <style>
                    h1 {
                      text-transform: uppercase;
                    }
                    p {
                      text-transform: capitalize;
                    }
                  </style>
                </head>

Tema: Css Font

21. Set the font family for the page to "Courier New", and the font family for h1 to "Verdana".


                <head>
                  <style>
                    body {
                      font-family: "Courier New";
                    }
                    
                    h1 {
                      font-family: Verdana;
                    }
                  </style>
                </head>

22. Show p elements as "bold" text.


                <head>
                  <style>
                    p {
                      font-weight: bold;
                    }
                  </style>
                </head>

Tema: Css Links

23. Set the color for links to "green".


                <head>
                  <style>
                    a {
                      color: green;
                    }
                  </style>
                </head>

24. Set the background color for visited and unvisited links to "lightblue", and the background color for the hover and active link states to "yellow".


                <head>
                  <style>
                  /* unvisited link */
                  a:link {
                    background-color: lightblue;
                  }
                  
                  /* visited link */
                  a:visited {
                    background-color: lightblue;
                  }
                  
                  /* mouse over link */
                  a:hover {
                    background-color: yellow;
                  }
                  
                  /* selected link */
                  a:active {
                    background-color: yellow;
                  }
                  </style>
                </head>

Tema: Css List

25. Set the list style for unordered lists to "square", and the list style for ordered lists to "upper-roman"


                <head>
                  <style>
                    ul { list-style-type: square; }
                    
                    ol { list-style-type: upper-roman;}
                  </style>
                </head>

26. With the list-style property: Set the unordered list marker to "img_marker.png", with a backup style of "circle", and display the markers inside the content flow..


                <head>
                  <style>
                    ul { list-style: circle inside url('img_marker.png'); }
                  </style>
                </head>

Tema: Css Tables

27. Set the border to "2px solid green" for table,th and td elements .


                <head>
                  <style>
                    table, th, td { border: 2px solid green;}
                  </style>
                </head>

28. Collapse the table borders into a single border.


                <head>
                  <style>
                    table {
                      border-collapse: collapse;
                    }
                    table, td, th {
                      border: 1px solid black;
                    }
                  </style>
                </head>

Tema: Css Display/Visibility

29. Hide the h1 element. It should still take up the same space as before.


                <head>
                  <style>
                   h1 { visibility: hidden;}
                  </style>
                </head>

30. Display the list items as inline elements..


                <head>
                  <style>
                  li { display: inline; }
                  </style>
                </head>

Tema: Css Positioning

31. Position the h1 element relative to the browser window. 50px from the top, and 50px from the right.


                <head>
                  <style>
                    h1 {
                      position: fixed;
                      top: 50px;
                      right: 50px;
                      color: red;
                    }
                  </style>
                </head>

32. Position the h1 element 50px from the left, and 100px from the top, relative to the HTML page.


                <head>
                  <style>
                    h1 {
                      position: absolute;
                      top: 100px;
                      left: 50px;
                      color: red;
                    }
                  </style>
                </head>

Tema: Css Overflow

33. Add a scrollbar to the div element.


                <head>
                  <style>
                    div {
                      background-color: #eee;
                      width: 200px;
                      height: 70px;
                      border: 1px dotted black;
                      overflow: scroll;
                    }
                  </style>
                </head>

34. Specify that the overflowing text in the div element should not be visible, not even with scrolling.


                <head>
                  <style>
                    div {
                      background-color: lightblue;
                      width: 200px;
                      height: 200px;
                      overflow: hidden;
                    }
                  </style>
                </head>

Tema: Css Align

35.Center align the div element using margins.


                <head>
                  <style>
                    div {
                      margin-left: auto;
                      margin-right: auto;
                      width: 300px;
                      background-color: #b0e0e6;
                    }
                  </style>
                </head>

36. Position the div element all the way to the right using absolute positioning.


                <head>
                  <style>
                    div {
                      position: absolute;
                      right: 0px;
                      width: 300px;
                      background-color: #b0e0e6;
                    }
                  </style>
                </head>

Tema: Css Combinator

37. Change the color of all p elements, that are descendants of div elements, to "red".


                <head>
                  <style>
                    div p { color: red; }
                  </style>
                </head>

38. Change the color of p elements, that are the adjacent (immediately following) sibling of a div element, to "red".


                <head>
                  <style>
                    div + p { color: red; }
                  </style>
                </head>

Tema: Css Pseudo-Classes

39. Set the background color for visited and unvisited links to "lightblue", and the background color for the hover and active link states to "yellow".


                <head>
                  <style>
                    /* unvisited link */
                    a:link {
                      background-color: lightblue;
                    }
                    
                    /* visited link */
                    a:visited {
                      background-color: lightblue;
                    }
                    
                    /* mouse over link */
                    a:hover {
                      background-color: yellow;
                    }
                    
                    /* selected link */
                    a:active {
                      background-color: yellow;
                    }
                  </style>
                </head>

40. Set the background color of input elements that are in focus (clicked or active), to "lightblue".


                <head>
                  <style>
                    input:focus { background-color: lightblue; }
                  </style>
                </head>
